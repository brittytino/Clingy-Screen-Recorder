## 🔮 Features

-   🖥️ Screen recording with options:
    -   🖥️ Screen only
    -   🔊 Screen with system audio
    -   🎙️ Screen with system audio + microphone
-   📺 Live preview of recordings
-   ⏯️ Easy-to-use interface for recording controls, including pause and resume
-   🛠️ Simple installation process
-   🌐 Cross-Browser Compatibility

## 🚀 Live Preview

You can view the live preview of the project [here](https://clingyflingy.vercel.app).

## 💻 Technologies Used

-   Next.js
-   Tailwind CSS
-   MediaStream API

## ⚙️ Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/sahilatahar/MediaMaster.git
    ```
2. Navigate to the project directory:
    ```bash
    cd MediaMaster
    ```
3. Install dependencies:
    ```bash
    npm install
    ```
4. Start the application:
    ```
    npm run dev
    ```
5. Open the application in your browser:
    ```
    http://localhost:3000
    ```
